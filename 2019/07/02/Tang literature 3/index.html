盛唐文学1

陈子昂（公元661～公元702，字伯玉，梓州射洪（今四川省射洪县）人，唐代文学家、诗人，初唐诗文革新人物之一。因曾任右拾遗，后世称陈拾遗。

以上是百度百科对陈子昂的介绍。陈子昂最出名的诗应该是《登幽州台歌》：

登幽州台歌
前不见古人，后不见来者。
念天地之悠悠，独怆然而涕下。

陈子昂随武则天的侄子武攸宜出征担任参谋，史书说武攸宜为人轻率，少谋略。陈子昂数次向武攸宜谏言不得采纳，反而被将为军曹，于是他写下了这首诗以表达他觉得自己怀才不遇，寂寞无聊的情绪。实际上，陈子昂并不很算怀才不遇，他的经历起码比初唐四杰好多了。陈子昂是富商家出身，最开始被武攸宜赏识，后来被推荐到武则天那里。他的官位虽不高，但性质上属于谏官，所以他常有机会向皇帝上谏。后人每当看到文人在作品里抒发自己不得志的情感的时候，会先入为主地认为他们的确是怀才不遇了，他们的皇帝/上司不采纳他们的意见是不对的。然而这只是他们自己的说法，事实上他们的谏言不见得就会起作用，这是我们读文学作品和历史时需要注意的。

（这一节讲了很多陈子昂的事迹，有点不明所以，我也忘了大部分。）

盛唐文学2

唐朝李家的政治势力属于关陇贵族，唐朝建立后，李世民需要整合、调节各贵族集团。广纳谏是他达到此目的的手段之一。唐朝初期本质上还是贵族政治，只不过统治阶级换了一拨人。然而到高宗武后时期，由于高宗想要摆脱当前贵族阶级对他皇权的限制，他和武后做了很多事情，其中包括科举“扩招”，重用没有背景的寒门士子，从而打破了原有的贵族统治的局面。这从客观上为唐朝的下层读书人提供了上升的通道。这就是盛唐文学到来的背景。不过这里说的下层也是相对的，因为在那个时代能读书并且读出点名堂的人，家里该都是有点背景的。

后世能看到的盛唐诗歌里，许多都是抒发不满的，比如李白的诗。为何这样一个强盛的时代会留下这么多不满的诗篇呢？这其实也是一个时代强盛的侧面表现，因为阳光越强烈，才显得影子越黑；而且他们有批判的自由。同时，也有很多看上去很出世、淡漠的诗，比如孟浩然的诗。

（这一节好像东讲讲西讲讲，我也总结不出来了。）
